Prediction of Dengue Fever

Developed and evaluated six ML models to predict dengue outbreaks using climatic data. Found that climate alone had weak correlation with case counts, emphasizing the need for social and biological features like hygiene and immunity. Published in the International Journal of Artificial Intelligence and Soft Computing.



Tools/Technologies used: Python, Machine Learning

Improving CLIP Training

Enhanced CLIP model training by implementing various combinations of advanced loss functions and optimizers. Achieved significant improvements in retrieval and zero-shot classification performance, with an average score of 19.12.

Multimodal Model for Predicting Lunch Calories

Developed a multimodal model to predict lunch calorie intake by integrating time-series, image, and demographic data for comprehensive analysis and prediction. Achieved an RMSRE score of 0.3374.



Tools/Technologies used: Python, Transformers, Deep Learning

Credit Card Fraud Detection

Developed a model for classifying credit card transactions with 99.999% accuracy and zero false negatives using data preprocessing and Apache Spark for optimized training.



Tools/Technologies used: Python, Machine Learning, Apache Spark
